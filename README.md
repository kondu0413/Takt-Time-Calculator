# Takt Time Calculator

製造現場向けのタクトタイム（Takt Time）計算機です。稼働時間、休憩時間、目標生産数を入力すると、必要なタクトタイムと1時間あたりの生産数を自動計算します。

## 機能

- **リアルタイム計算**: 入力値を変更すると即座に結果が更新されます
- **データ永続化**: 入力値はブラウザのlocalStorageに保存され、次回アクセス時も保持されます
- **モバイル対応**: スマートフォンやタブレットでも快適に使用できるレスポンシブデザイン
- **ダークモード対応**: システムの設定に応じて自動的に切り替わります
- **PWA対応**: オフラインでも動作し、ホーム画面に追加してアプリのように使用できます

## 計算式

1. **稼働可能時間（分）** = (稼働時間 × 60) - 休憩時間
2. **タクトタイム（秒）** = (稼働可能時間 × 60) ÷ 目標生産数
3. **1時間あたりの必要数** = 3600 ÷ タクトタイム

## 開発環境のセットアップ

### 前提条件

- Node.js 18.x 以上
- npm または yarn

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリケーションを確認できます。

### ビルド

```bash
npm run build
```

### 本番環境での起動

```bash
npm start
```

## 技術スタック

- **Framework**: Next.js 16.1.1 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4.x
- **UI**: React 19.2.3
- **PWA**: Service Worker + Web App Manifest

## PWA機能

このアプリはProgressive Web App (PWA)として動作します。

### インストール方法

1. **モバイル（iOS/Android）**
   - Safariまたはブラウザでアプリを開く
   - 「ホーム画面に追加」を選択
   - アプリアイコンがホーム画面に追加されます

2. **デスクトップ（Chrome/Edge）**
   - アプリを開く
   - アドレスバーの右側にあるインストールアイコンをクリック
   - 「インストール」を選択

### オフライン対応

- 一度アプリを読み込むと、オフラインでも使用できます
- 計算機能はすべてブラウザ上で動作するため、インターネット接続は不要です
